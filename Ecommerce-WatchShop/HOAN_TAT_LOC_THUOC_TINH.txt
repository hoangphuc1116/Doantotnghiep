🔧 KỸ THUẬT:

1. CONTROLLER:
   ```csharp
   public async Task<IActionResult> Index(
       List<int>? attributeValues = null)
   {
       if (attributeValues != null && attributeValues.Any())
       {
           products = products.Where(p => 
               p.SanPhamThuocTinhs.Any(pt => 
                   attributeValues.Contains(pt.MaGiaTri)));
       }
   }
   ```

2. VIEW:
   ```razor
   @foreach (var attr in attributes)
   {
       <div class="sidebar-single-widget">
           <h6>@attr.TenThuocTinh</h6>
           @foreach (var value in attr.GiaTriThuocTinhs)
           {
               <input type="checkbox" 
                      name="attributeValues" 
                      value="@value.MaGiaTri">
               <label>@value.GiaTri</label>
           }
       </div>
   }
   ```
